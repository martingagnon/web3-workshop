### Web3 Workshop

# Setup
```
sh ./setup.sh
```

# Run tests
```
yarn hardhat test
```

# Start interactive console
```
yarn hardhat console
```

# Start node
```
yarn hardhat node
```

# Deploy demo contract
```
yarn hardhat run ./scripts/deploy-contract.ts --network localhost
```
 
# Start webapp
```
cd webapp
yarn dev
```
